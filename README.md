## Funcionalidades

**Painel Admin**
- Dashboard com estatísticas
- Criar e gerenciar eventos
- Upload de fotos (drag & drop)
- Visualizar seleções dos clientes
- Aprovar entregas

**Galeria do Cliente**
- Login por e-mail e senha do evento
- Visualização em grid
- Filtros por categoria
- Busca por código da foto
- Favoritar e selecionar fotos
- Lightbox com navegação
- Finalizar seleção

## Como rodar localmente

```bash
npm install
npm start
```

Acesse: http://localhost:3000
